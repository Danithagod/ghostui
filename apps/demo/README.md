# The Cauldron Emporium - GhostUI Demo

A demo e-commerce application showcasing the GhostUI component library. This spooky potion shop demonstrates how to integrate and use GhostUI components in a real-world application.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.

## Features

- Browse magical potions, elixirs, and cursed items
- Search and filter products
- Shopping cart functionality
- Multi-step checkout flow
- Theme switching (dark/light mode)
- Ambient visual effects
